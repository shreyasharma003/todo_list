# To-Do List Web App ✅📝

A simple and interactive To-Do List web application built using **HTML**, **CSS**, and **JavaScript**.

## 🚀 Features
- ➕ Add new tasks
- ✅ Priority wise tasks
- 🗑️ Delete tasks
- 💾 Data stored temporarily (can be extended with localStorage)

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)
